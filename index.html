Low Cost IOT Based Smart Monitoring System

Project Overview
[IoT ESP32 System]
Developers Gayathri S, Anitha S Prasad, et al.
Platform ESP32 Microcontroller
Sensors PIR, DHT11, Vibration
Connectivity Wi-Fi, Telegram Bot
Data Cloud ThingSpeak

The Low Cost IOT Based Smart Monitoring System is a cost-effective surveillance and environmental monitoring prototype designed to enhance home and industrial security. Built using the Espressif ESP32 microcontroller, the system integrates motion-triggered image capture with real-time environmental data logging.

Description

The system utilizes a Passive Infrared (PIR) sensor to detect human movement. Upon detection, the ESP32-CAM module captures a high-resolution image and securely transmits it to a user's Telegram account via Wi-Fi. This ensures that the user receives instant visual alerts of any intrusion.

Core Features
  • Motion Detection: Uses PIR and vibration sensors to trigger alerts.
  • Instant Messaging: Sends captured images and alerts via a Telegram Bot.
  • Environmental Sensing: Measures temperature and humidity using a DHT11 sensor.
  • Cloud Analytics: Uploads sensor data to ThingSpeak for remote analysis and historical tracking.
  • Energy Management: Includes a relay module to control external devices like light bulbs, contributing to sustainable energy use.
Performance

According to testing results, the system demonstrates a motion detection response time of approximately 12 seconds, while the end-to-end notification (image capture to Telegram message) takes roughly 5 seconds under stable Wi-Fi conditions.
